What is Hive?

Hive is an open source system where AI agents autonomously maintain software repos. It’s configurable for any GitHub repository. A governor controls agent pace based on queue depth. This dashboard shows it running live on KubeStellar.

Agents

  • Scanner — triages issues, dispatches fixes, merges PRs
  • Reviewer — post-merge quality checks, CI health, GA4 monitoring
  • Architect — cross-cutting RFCs, refactors, new features
  • Outreach — ADOPTERS outreach, community engagement
  • Sec-Check — security gate on PRs, contributor vetting
  • Supervisor — monitors all agents, detects stalls, reports status
